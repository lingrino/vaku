## vaku folder copy

Recursively copy all secrets in source folder to destination folder

### Synopsis

Recursively copy all secrets in source folder to destination folder

```
vaku folder copy <src> <dst> [flags]
```

### Examples

```
vaku folder copy secret/foo secret/bar
```

### Options

```
    , --all-versions   copy all versions of the secret (KV v2 only)
```
