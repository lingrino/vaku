## vaku folder move

Recursively move all secrets in source folder to destination folder

### Synopsis

Recursively move all secrets in source folder to destination folder

```
vaku folder move <source folder> <destination folder> [flags]
```

### Examples

```
vaku folder move secret/foo secret/bar
```
