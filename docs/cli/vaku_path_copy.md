## vaku path copy

Copy a secret from a source path to a destination path

### Synopsis

Copy a secret from a source path to a destination path

```
vaku path copy <src> <dst> [flags]
```

### Examples

```
vaku path copy secret/foo secret/bar
```

### Options

```
    , --all-versions   copy all versions of the secret (KV v2 only)
```
