Vaku

Vaku is a CLI and Go API that extends the official Hashicorp Vault CLI and API with useful high-level functions such as the ability to copy, move, and search Vault paths and folders. The CLI and API are purpose-built to deal with the actual secret content of paths and folders, and they therefore do not make use of or reveal Vault features like metadata and wrapping. Use the tools as you see fit and file issues for anything you think could be improved. Thanks!
